# 🏺 Seeds of the Nile: Instructions

Welcome! This is a simple puzzle game where you match ancient Egyptian seeds to the crops they grow. You don't need any programming knowledge to play this on your Windows computer.

Follow these easy steps to get started:

### 1. Download the Game
*   Go to the project page on GitHub (the link where you found this).
*   Look for the green **"<> Code"** button near the top right.
*   Click it and select **"Download ZIP"**.

### 2. Extract the Files
*   Once the download is finished, find the file (usually in your **Downloads** folder) named `test_puzzle-main.zip` (or similar).
*   **Right-click** the file and select **"Extract All..."**.
*   A window will pop up; just click the **"Extract"** button at the bottom.
*   A new folder will open automatically containing the game files.

### 3. Play the Game
*   In the folder you just opened, look for a file named **`index.html`**. 
    *   *Note: If you don't see the `.html` part, it will just be named `index` and have a web browser icon (like Chrome, Edge, or Firefox).*
*   **Double-click** that file.
*   The game will open in your web browser!

---

### How to Play
*   **The Goal:** You will see a scene from ancient Egypt with one item missing. Your job is to figure out which seed grows that missing item!
*   **Choose a Seed:** Look at the "Seed Tray" at the bottom.
*   **Two ways to answer:**
    1.  **Click and Click:** Click on a seed to select it (it will glow), then click on the empty "question mark" slot in the scene.
    2.  **Drag and Drop:** Click and hold a seed, drag it over to the question mark slot, and let go.
*   **Progress:** There are 3 different scenes to master. Can you get a perfect score?

### Troubleshooting
*   **Images aren't showing up?** Make sure you "Extracted" the files in Step 2. If you just double-click the ZIP file and try to run `index.html` from inside it without extracting, the game won't be able to find its pictures.
*   **Screen too small?** You can zoom in or out using your browser's zoom (usually `Ctrl` and the `+` or `-` keys).

Enjoy your journey to the banks of the Nile! ☀️🌾
